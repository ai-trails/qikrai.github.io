## 🏗 **Local Development**
To run the website locally, install **Hugo** and follow these steps:

### **1️⃣ Clone the Repository**
```sh
git clone --recurse-submodules https://github.com/ai-trails/qikrai.github.io.git
cd qikrai.github.io
hugo server -D
```
